Claude Codeは日本語で指示するだけでコードを自動生成するAIツールです。この記事では、起動から基本操作、得意・不得意まで初心者向けに解説します。

## Claude Codeの基本操作

Claude Codeは「日本語で指示を出す → AIが実行する」というシンプルな流れで動きます。プログラミングの知識がなくても使えます。

## 起動と終了

```bash
# 起動
claude

# 終了（Claude Codeの中で）
/exit
```

## 日本語で指示する例

### ファイルを作る

```
Todoアプリのindex.htmlを作って
```

### コードを読んでもらう

```
このプロジェクトの構成を説明して
```

### バグを直す

```
エラーが出てるから直して
```

### リファクタリング

```
このコードをもっときれいに書き直して
```

これだけです。難しいコマンドを覚える必要はありません。

## 知っておくと便利な操作

### スラッシュコマンド

Claude Codeにはいくつかの特別なコマンドがあります：

| コマンド | 機能 |
|----------|------|
| `/help` | ヘルプを表示 |
| `/clear` | 会話履歴をクリア |
| `/compact` | 会話を要約して圧縮 |
| `/exit` | 終了 |

### ファイルを指定して質問する

```
src/App.tsx を読んで、何をしているか説明して
```

ファイルパスを含めると、そのファイルを読んだうえで回答してくれます。

### 複数の指示を一度に出す

```
1. src/components/ に Header コンポーネントを作って
2. App.tsx で Header を読み込んで
3. CSSはTailwindで書いて
```

番号付きで箇条書きにすると、順番に実行してくれます。

## Claude Codeが得意なこと

- **新規ファイルの作成** — 「〜を作って」で一発
- **既存コードの修正** — プロジェクト全体を理解したうえで修正
- **エラーの解決** — エラーメッセージを貼るだけで原因と対処を教えてくれる
- **テストの作成** — 「テストを書いて」で自動生成
- **ドキュメント作成** — READMEやコメントの自動生成

## Claude Codeが苦手なこと

- **最新のライブラリ情報** — 学習データの時点までの知識しかない
- **大規模プロジェクトの全体把握** — ファイル数が多すぎると見落としがある
- **デザインの判断** — 「かっこよくして」は苦手。具体的に指示する方がよい

## 実践：Webページを作ってみよう

実際にClaude Codeでシンプルなプロフィールページを作ってみましょう。

```bash
mkdir profile-page
cd profile-page
claude
```

起動したら：

```
シンプルなプロフィールページを作って。
名前は「田中太郎」、職業は「エンジニア」、
趣味は「読書とプログラミング」にして。
レスポンシブ対応で、モダンなデザインにして。
```

数秒〜数十秒でHTMLファイルが生成されます。ブラウザで開いて確認してみてください。

気に入らない部分があれば：

```
背景色をもう少し暗くして
フォントサイズを大きくして
```

と追加で指示するだけで修正してくれます。

## まとめ

- Claude Codeは **日本語で指示するだけ** で使える
- スラッシュコマンドで会話の管理ができる
- 得意・不得意を理解して使い分けるのがコツ
