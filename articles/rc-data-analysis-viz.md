Claude Codeを使えば、CSVデータの分析やグラフ作成を日本語の一言で自動化できます。Python/Node.jsを使った実践的なデータ可視化の方法を解説します。

## この記事でやること

Claude Codeを使えば、CSVデータの分析とグラフ作成を「このCSVを分析して」の一言で自動化できます。Excelより速く、コードも残る実践的な方法を解説します。

## なぜClaude Codeでデータ分析？

- 「このCSVをグラフにして」で完成
- Excel関数を覚えなくていい
- 複雑な集計も日本語で指示するだけ
- 再利用可能なスクリプトが残る

## 準備

Pythonを使います（Node.jsでも可能ですが、データ分析はPythonが得意）：

```bash
mkdir data-analysis
cd data-analysis
claude
```

## 実践1：売上データの分析

### CSVを用意

```
sales.csvというサンプルデータを作って。
列：日付, 商品名, カテゴリ, 数量, 単価, 売上
100行くらい。2024年1月〜12月のデータで。
商品カテゴリは「食品」「飲料」「日用品」の3つ。
```

### 基本集計

```
sales.csvを分析して。

1. 月別の売上合計
2. カテゴリ別の売上構成比
3. 最も売れた商品トップ10
4. 前月比の増減率

結果をコンソールに表形式で出力して。
```

### グラフ作成

```
以下のグラフを作成してPNGで保存して：

1. 月別売上の折れ線グラフ
2. カテゴリ別売上の円グラフ
3. 商品別売上トップ10の棒グラフ

日本語フォント対応で。matplotlib使って。
```

## 実践2：アクセスログの分析

```
Webサイトのアクセスログ（CSV）を分析するスクリプトを作って。

分析項目：
- 時間帯別アクセス数（ヒートマップ）
- ページ別PVランキング
- 日別のユニークユーザー数推移
- リファラー別の流入割合

グラフをHTMLレポートとして出力して。
ブラウザで開ける形式で。
```

## 実践3：アンケート結果の可視化

```
アンケート結果のCSVを分析して。
5段階評価の項目が10個ある想定。

分析内容：
- 各項目の平均スコアをレーダーチャートで
- 回答分布をヒストグラムで
- 自由記述をワードクラウドで（あれば）

全てのグラフを1つのHTMLページにまとめて。
```

## Node.jsでやる場合

Pythonが使えない環境ではNode.jsでも可能です：

```
Node.jsでCSVファイルを読み込んで分析するスクリプトを作って。
Chart.jsを使ってHTMLにグラフを埋め込んで。
ブラウザで開けるレポートとして出力。
```

## 便利なプロンプト集

```
# データの概要を把握
「このCSVの基本統計量を出して」

# 異常値の検出
「外れ値がないか確認して。あれば教えて」

# 相関分析
「各列の相関係数を計算してヒートマップで見せて」

# 予測
「過去12ヶ月のデータから来月の売上を予測して」
```

## まとめ

- Claude Codeなら「このCSVを分析して」でOK
- グラフ作成もPNG/HTMLで自動出力
- PythonでもNode.jsでも対応可能
- 繰り返し使えるスクリプトが残る
