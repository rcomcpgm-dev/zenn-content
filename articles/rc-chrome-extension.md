## この記事でやること

Claude Codeを使って、**実用的なChrome拡張機能**をゼロから作ります。今回作るのは「Webページの文字数をカウントする拡張機能」です。

## Chrome拡張機能とは

ブラウザに追加機能をつけるプログラムです。AdBlockやDeepLなど、普段使っている拡張機能と同じものを自分で作れます。

## 作っていこう

```bash
mkdir chrome-word-counter
cd chrome-word-counter
claude
```

### ステップ1：基本構成を作る

```
Chrome拡張機能を作りたい。Manifest V3で。
機能：現在のWebページの文字数をカウントして表示する。
ポップアップUIで結果を表示。

以下のファイルを作って：
- manifest.json
- popup.html
- popup.js
- content.js
- styles.css
```

Claude Codeがすべてのファイルを生成してくれます。

### ステップ2：Chromeに読み込む

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. プロジェクトフォルダを選択

ツールバーにアイコンが表示されます。

### ステップ3：機能を拡張する

```
選択したテキストの文字数もカウントできるようにして。
ポップアップに「ページ全体: ○○文字」「選択中: ○○文字」の2行で表示。
```

```
文字数だけでなく、単語数、行数、読了目安時間も表示して。
日本語は400文字/分、英語は200words/分で計算。
```

### ステップ4：デザインを整える

```
ポップアップのデザインをきれいにして。
幅300px、角丸、影つき。数字は大きく太字で表示。
ダークモード対応もお願い。
```

## 他にも作れる拡張機能の例

| 拡張機能 | 指示の例 |
|----------|---------|
| タブ管理 | 「開いているタブを一覧表示してまとめて閉じられる拡張を作って」 |
| スクリーンショット | 「ページのスクショを撮ってクリップボードにコピーする拡張」 |
| リーディングリスト | 「今のページを後で読むリストに保存する拡張」 |
| カラーピッカー | 「ページ上の色をHEXコードで取得する拡張」 |
| 翻訳 | 「選択テキストをポップアップで翻訳する拡張」 |

## Chrome Web Storeに公開する

1. Chrome Web Store Developer Dashboardに登録（初回$5）
2. ZIPファイルにまとめてアップロード
3. 審査を待つ（数日）

```
このプロジェクトをChrome Web Storeに公開するための
ZIPファイルを作って。不要なファイルは除外して。
```

## まとめ

- Chrome拡張機能は **manifest.json + HTML + JS** だけで作れる
- Claude Codeなら日本語の指示だけで完成
- 「デベロッパーモード」でローカルテスト
- Web Storeに公開すれば世界中の人が使える
