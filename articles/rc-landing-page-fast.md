## この記事でやること

クライアントワークや自分のサービス告知に使える**ランディングページ（LP）**を、Claude Codeで爆速で作ります。

## LPとは

1ページ完結の営業ページ。商品紹介、サービス申し込み、メルマガ登録など、1つの目的に特化したページです。

## LP1本の相場

| レベル | 相場 |
|--------|------|
| テンプレ流用 | 3〜5万円 |
| オリジナルデザイン | 10〜30万円 |
| コーディング込み | 20〜50万円 |

Claude Codeを使えば、オリジナルデザインのLPを数時間で作れます。

## 実際に作ってみる：パーソナルジムのLP

```bash
mkdir gym-lp
cd gym-lp
claude
```

### ステップ1：全体構成を指示する

```
パーソナルジムのランディングページを作って。
HTML + Tailwind CSS（CDN）で1ファイル完結。

セクション構成：
1. ファーストビュー（キャッチコピー + CTA）
2. こんな悩みありませんか？（3つ）
3. 選ばれる3つの理由
4. トレーナー紹介
5. 料金プラン（3つ）
6. お客様の声（3件）
7. よくある質問（アコーディオン）
8. アクセス
9. お問い合わせフォーム + CTA

デザイン：高級感のある黒×ゴールド。
```

### ステップ2：テキストを調整

```
キャッチコピーを「3ヶ月で人生が変わる。完全マンツーマンのパーソナルジム」に。
CTAボタンは「無料カウンセリングを予約する」にして。
```

### ステップ3：アニメーション追加

```
スクロールに合わせて各セクションがフェードインするアニメーション。
IntersectionObserverで。ライブラリは使わない。
```

### ステップ4：レスポンシブ対応

```
スマホ表示を最適化して。
料金プランは横並びから縦並びに。フォントサイズも調整。
```

## テンプレを量産するコツ

業種ごとのLPテンプレを作っておくと受注が速くなります：

```
美容院のLPを作って。構成はジムLPと同じで、
テキストとデザイン（ピンク×白の清潔感あるデザイン）を変えて。
```

業種を変えるだけで新しいLPが数分で出来上がります。

## 副業としてのLP制作

### メリット
- 1案件5〜30万円の高単価
- Claude Codeで制作時間を大幅短縮
- 在宅で完結
- クラウドソーシング（ランサーズ、ココナラ）で営業

### 案件獲得のコツ
- ポートフォリオにLP実績を3〜5個載せる（ダミーでOK）
- 「最短3日で納品」をアピール
- 初回は相場より安く受けて実績を作る

## まとめ

- Claude Codeなら日本語の指示だけでLPが作れる
- 1ページ完結のHTMLなので技術的にもシンプル
- 副業として月5〜20万円の収入も狙える
