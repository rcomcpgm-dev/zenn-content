Claude Codeを使えば、React + Tailwind CSSのモダンなポートフォリオサイトを30分で構築できます。転職・フリーランス営業に使える実践的な作り方を解説します。

## この記事でやること

Claude Codeを使えば、転職やフリーランス営業に使えるポートフォリオサイトを30分で作れます。React + Tailwind CSSでモダンなサイトを構築する手順を解説します。

## 完成イメージ

- 自己紹介セクション
- スキル一覧
- 作品（プロジェクト）紹介
- お問い合わせフォーム
- レスポンシブ対応（スマホでもきれいに表示）

## 準備

- Node.jsインストール済み
- Claude Code使える状態

```bash
mkdir my-portfolio
cd my-portfolio
claude
```

## ステップ1：プロジェクトの雛形を作る（5分）

Claude Codeに以下を指示します：

```
React + Vite + TypeScript + Tailwind CSSでポートフォリオサイトのプロジェクトを作って。
セットアップまで完了させて。
```

AIがプロジェクトの初期化、依存パッケージのインストール、Tailwindの設定まで全部やってくれます。

## ステップ2：メインページを作る（10分）

```
ポートフォリオのメインページを作って。以下のセクションを含める：

1. ヒーローセクション（名前「山田太郎」、肩書き「フルスタックエンジニア」、
   一言「テクノロジーで課題を解決する」）
2. Aboutセクション（自己紹介文を適当に入れて）
3. Skillsセクション（React, TypeScript, Node.js, Python, AWS, Docker）
4. Projectsセクション（ダミーで3つ。画像はplaceholderで）
5. Contactセクション（お問い合わせフォーム。名前、メール、メッセージ）

デザインはモダンでシンプル。ダークモード風。
スムーズスクロールで各セクションに移動できるナビゲーションバーもつけて。
```

これだけで、かなり見栄えの良いページが出来上がります。

## ステップ3：デザインを調整する（10分）

生成されたサイトを `npm run dev` で確認しながら、気になる点を修正します：

```
ヒーローセクションの背景にグラデーションを追加して。
紫から青のグラデーションで。
```

```
Projectsのカードにホバーエフェクトを追加して。
ふわっと浮き上がる感じで。
```

```
スマホで見たときにナビゲーションがハンバーガーメニューになるようにして
```

こんな感じで日本語で微調整するだけです。

## ステップ4：デプロイする（5分）

無料でデプロイできるVercelを使います：

```
このプロジェクトをVercelにデプロイできる状態にして。
ビルドが通ることを確認して。
```

その後ターミナルで：

```bash
npm install -g vercel
vercel
```

数回Enterを押すだけでデプロイ完了。URLが発行されます。

## カスタマイズのヒント

### 自分の情報に変更する

```
名前を「鈴木花子」に、肩書きを「Webデザイナー」に変更して。
スキルをFigma, Photoshop, HTML, CSS, JavaScript, WordPressに変更して。
```

### アニメーションを追加する

```
各セクションにスクロールアニメーション（フェードイン）を追加して。
framer-motionを使って。
```

## よくある質問

**Q. 本当に30分でできる？**
A. シンプルなものなら十分可能。凝ったデザインにするなら1〜2時間。

**Q. プログラミング知識ゼロでも大丈夫？**
A. はい。Claude Codeが全部書いてくれます。

**Q. 維持費はかかる？**
