CLI（コマンドラインインターフェース）とは、キーボードで文字を打ってパソコンに指示を出す方法です。この記事では、黒い画面に触れたことがない完全初心者向けに、CLIの基本をやさしく解説します。

## この記事は誰のため？

- パソコンは使えるけど「黒い画面」は触ったことがない
- プログラミングに興味はあるけど、何から始めればいいかわからない
- 最近話題の「Claude Code」を使ってみたいけど、CLIが必要らしくて困っている

**安心してください。** この記事を読めば、CLIが「ただの便利な道具」だとわかります。

## CLIとは？

**CLI = Command Line Interface（コマンドライン・インターフェース）**

ひとことで言うと、**キーボードで文字を打ってパソコンに指示を出す方法**です。

普段みなさんが使っているのは **GUI（Graphical User Interface）** ——マウスでアイコンをクリックしたり、ボタンを押したりする操作ですよね。

| | GUI | CLI |
|---|---|---|
| 操作方法 | マウスでクリック | キーボードで文字入力 |
| 見た目 | カラフルなボタンやアイコン | 黒い画面に白い文字 |
| 例え | レストランのメニュー表 | シェフに直接注文 |

**CLIはシェフに直接注文するようなもの。** メニューにない料理もお願いできるし、細かいカスタマイズも自由自在です。

## なぜ「黒い画面」は怖く見えるのか

怖く感じる理由は主に3つです：

### 1. 何を打てばいいかわからない
GUIなら画面にボタンがあるので「押せばいい」とわかります。CLIは白紙の状態から文字を打つので、最初は戸惑います。

→ **でも大丈夫。** 使うコマンドは最初はたった数個だけです。

### 2. 間違えたら壊れそう
「変なコマンドを打ったらパソコンが壊れるのでは？」という不安。

→ **ほぼ壊れません。** 普通のコマンドでパソコンが壊れることはまずありません。間違えたら「そんなコマンドないよ」とエラーが出るだけです。

### 3. 映画のハッカーっぽくて難しそう
映画で見る高速タイピングのイメージが強すぎます。

→ **現実は全然違います。** ゆっくり1行ずつ打てばOKです。コピペもできます。

## 実際にCLIを見てみよう

Windowsの場合、以下の手順で開けます：

1. キーボードの **Windowsキー** を押す
2. **「cmd」** と入力する
3. **「コマンドプロンプト」** が出てくるのでクリック

黒い画面が開きましたか？ 怖くないですよね。ただの入力欄です。

試しに以下を打ってEnterを押してみてください：

```
echo こんにちは
```

画面に「こんにちは」と表示されたはずです。**おめでとうございます、これがCLIです！**

## よく使う基本コマンド5つ

最初に覚えるのはこれだけで十分です：

```bash
# 今いる場所（フォルダ）を確認する
cd

# フォルダの中身を見る
dir

# フォルダを移動する
cd Desktop

# 画面をきれいにする
cls

# こんにちは、と表示する
echo こんにちは
```

たった5つ。これだけで「CLIを使える人」です。

## なぜ今CLIを覚えるべきなのか

答えは **AI** です。

2025年以降、AIを使った開発ツールが爆発的に増えています。その中でも特に注目されているのが **Claude Code** というツール。

Claude Codeは **ターミナル（CLI）上で動くAIアシスタント** で、日本語で話しかけるだけでプログラミングをしてくれます。

```
あなた：「Todoアプリを作って」
Claude Code：「はい、作成します」→ 数分で完成
```

こんな魔法のようなツールですが、使うにはCLIの基礎知識が必要です。だからこそ、今このタイミングで覚える価値があるのです。

- CLIは **キーボードでパソコンに指示を出す方法**
- **怖くないし、壊れない**
